# cloudboard
Copy here, paste everywhere!

Playing with clipboardy and Pusher.

```
# clone this project
cd cloudboard
npm install
npm start
```

# Requirements
- Nodejs
- Pusher account https://pusher.com/

# Supports
- MacOS
- Windows
- Linux
